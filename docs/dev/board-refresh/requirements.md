# ブロック消去・補充 要件

## 対象 issue

- #8 ゲーム進行: 正解後のブロック消去・補充を実装する

## 目的

正解後にブロックを消し、次に遊べる盤面へ更新する。

## 要件

- 正解した 3 ブロックに消去状態を付ける。
- 成功演出中は盤面入力を止める。
- 成功演出後、同じレベルの新しい盤面へ更新する。
- 更新後の盤面も最低保証正解数を満たす。
- 補充は初期実装では再生成方式に寄せる。
- 古い refresh timer が後続盤面を上書きしないようにする。
- 既に queue 済みの古い refresh callback も後続盤面を上書きしないようにする。
- 空きマスが残らない。
- 数字は `1` から `18` の範囲に収める。

## 非スコープ

- 物理的な落下アニメーション。
- 連鎖演出。
- スコア加算。

## 完了条件

- 正解後に選択ブロックが消える状態になる。
- その後、新しい数字盤面が表示される。
- 継続して次の式を探せる。
- 古い盤面更新 timer は新しい更新予約時や再描画時に解除され、queue 済み callback も無効化される。
