# 盤面 UI 要件

## 対象 issue

- #6 UI: 盤面と選択中の式プレビューを表示する

## 目的

生成された盤面を画面に表示し、ドラッグ選択中のブロックと式プレビューを見えるようにする。

## 要件

- レベル設定から盤面サイズを決める。
- 生成された数字ブロックをグリッド表示する。
- 各ブロックは入力 controller が参照できる `data-cell-id` を持つ。
- 選択中のブロックを強調表示する。
- 選択中に式プレビューを表示する。
- 3 個選択されたら、成立式または未成立の式を表示する。
- スコア競争ではなく、盤面と式を主役にする。
- スマホ幅でも数字や式がはみ出さない。

## 非スコープ

- 正解時の消去アニメーション。
- ブロックの補充。
- 段階ヒント。

## 完了条件

- `5 x 5` の盤面が表示される。
- ブロックをドラッグすると選択中のブロックが強調される。
- 選択中に `5 ± 7 = ?` のようなプレビューが表示される。
- 成立時に `5 + 7 = 12` のような式が表示される。
