# 盤面生成 要件

## 対象 issue

- #3 盤面生成: 最低保証正解数を満たすボードを生成する

## 目的

完全ランダムではなく、難易度ごとの最低保証正解数を満たす盤面を生成する。

## 要件

- `w x h` の盤面を生成できる。
- 各セルは `row`、`col`、`value` を持つ。
- 数字は `1` から `18` の範囲に収める。
- `0` は生成しない。
- レベル設定の `guaranteedAnswerCount` 以上の正解候補を保証する。
- LV1 では保証正解を左から右、上から下に配置する。
- LV3 以降では保証正解に右から左、下から上も混ぜられる。
- 生成後に盤面をスキャンし、最低保証正解数を満たすか確認する。
- seed を渡した場合は再現可能にする。

## 非スコープ

- プレイヤー選択の妥当性判定。
- DOM 表示。
- 正解後の消去・補充。

## 完了条件

- LV1 で最低 4 個の保証正解を持つ盤面を生成できる。
- LV2 で最低 4 個、LV3 で最低 3 個、LV4 で最低 2 個の設定を扱える。
- テストで `0` が生成されないことを確認できる。
- テストで seed による再現性を確認できる。
