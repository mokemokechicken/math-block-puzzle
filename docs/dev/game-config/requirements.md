# ゲーム設定 要件

## 対象 issue

- #2 ゲーム設定: レベル・数字範囲・演算ルールを定義する

## 目的

盤面生成、正解判定、UI が同じゲーム設定を参照できるようにする。

## 要件

- 数字範囲は `1` から `18`。
- `0` は初期実装では出さない。
- 選択ブロック数は 3 個。
- 最後のブロックを答えとして扱う。
- LV1 は足し算のみ。
- LV2 以降は足し算と引き算を扱う。
- 正解判定方向は LV1 から全方向を許可する。
- 保証正解の配置方向は、正解判定方向とは別に定義する。
- LV1/LV2 は保証正解を左から右、上から下に寄せる。
- LV3 以降は保証正解に右から左、下から上も混ぜられる。

## 非スコープ

- 盤面生成の実装。
- 正解判定の実装。
- UI へのレベル選択表示。

## 完了条件

- レベル設定が JavaScript から参照できる。
- テストで数字範囲、演算、方向、保証正解数を確認できる。
- 判定可能方向と保証配置方向が別々に表現されている。
