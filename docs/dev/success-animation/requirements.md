# 成功演出 要件

## 対象 issue

- #7 成功演出: 消えたブロックから式が浮き上がるアニメーションを実装する

## 目的

正解時に、選んだ数字と成立した式が視覚的につながる演出を追加する。

## 要件

- 正解したブロックを一瞬光らせる。
- 選択ブロックの位置から成立式を浮き上がらせる。
- 式は少し残ってからフェードアウトする。
- 不正解時には成功演出を出さない。
- 演出は短くし、次の操作を邪魔しない。
- reduced motion に配慮する。

## 非スコープ

- ブロックの消去。
- ブロック補充。
- 連続正解演出。

## 完了条件

- 正解時に `5 + 7 = 12` のような式が盤面上に浮く。
- 正解したブロックに成功状態の class が付く。
- 式表示は一定時間後に消える。
