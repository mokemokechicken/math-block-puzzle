# ドメイン用語

## ゲーム全体

- 足し算・引き算ブロックパズル: 小学校低学年向けに、数字の並びから成立する式を見つける Web パズルゲーム。
- 盤面: 数字ブロックが格子状に並ぶ領域。現行実装ではレベルにより `4 x 4` または `5 x 5`。
- 数字ブロック: 盤面上の 1 マス。`row`、`col`、`value`、`id` を持つ。
- セル: 実装上の数字ブロック単位。

## 式と判定

- 選択: プレイヤーが縦、横、または L 字型になぞった 3 ブロックのセル列。
- 選択長: 1 回の判定に使うセル数。現行実装では常に 3。
- 左辺: 選択された 3 ブロックのうち、最後以外の 2 ブロック。
- 答えブロック: 選択された 3 ブロックの最後のブロック。
- 成立式: `a + b = c` または `a - b = c` として成立する選択。
- 正解候補: 現在の盤面に存在する成立式。
- 保証正解: 盤面生成時に、最低正解数を満たすため意図的に埋め込む成立式。
- 偶然正解: 保証正解として埋め込んでいないが、ランダム補充後の盤面スキャンで成立する式。

## 方向

- 正解判定方向: プレイヤーが見つけた式を正解として受け付ける方向。
- 保証配置方向: 盤面生成時に保証正解を埋め込む方向。
- 左から右: `row` 差分 0、`col` 差分 +1。
- 右から左: `row` 差分 0、`col` 差分 -1。
- 上から下: `row` 差分 +1、`col` 差分 0。
- 下から上: `row` 差分 -1、`col` 差分 0。

## UX

- 式プレビュー: 選択中の数字を式として表示する領域。
- 成功演出: 正解したブロックを光らせ、成立式を盤面上に浮き上がらせる演出。
- 段階ヒント: 無操作時間に応じて、正解候補の一部から式全体まで段階的に示す補助表示。
- 通常モード: 5 問正解、または正解候補がなくなるまで問題を解く既定モード。
- 1分アタック: 60 秒間で正解を重ね、スコアを伸ばすゲームモード。
- 累積倍率: 1分アタックで、10 秒以内の連続正解によって上乗せされる得点倍率。
- 直近獲得点: 1分アタックで、最後に正解した 1 問により加算された点数。
